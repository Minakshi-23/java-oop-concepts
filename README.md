# Java OOP Concepts

This repository contains Java programs demonstrating core Object-Oriented Programming concepts.

## Topics Covered

- Inheritance
- Polymorphism
- Abstraction
- Interface
- Exception Handling

## Files

- InheritanceExample.java
- PolymorphismExample.java
- AbstractionExample.java
- InterfaceExample.java
- ExceptionHandlingExample.java

## Purpose

The goal of this repository is to understand and practice the fundamental OOP concepts in Java.

These programs are useful for beginners and students preparing for interviews.
